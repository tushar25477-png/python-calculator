# 🧮 Python Scientific Calculator

A simple command-line scientific calculator built using Python.

## 🚀 Features
- Basic operations (Add, Subtract, Multiply, Divide)
- Trigonometric functions
- Inverse Trigonometry
- Logarithm & Exponential
- Power & Factorial
- Prime number check
- HCF (GCD)
- Average calculation
- Memory feature (`ans`)
- Continuous loop (like real calculator)

## 🛠️ Technologies Used
- Python
- math module

## ▶️ How to Run
```bash
python calculator.py
